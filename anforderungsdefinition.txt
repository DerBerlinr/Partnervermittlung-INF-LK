1. Zielbestimmung:
    1.1 Programm muss Benutzer in eine Liste hinzufügen und passende Partner in der Liste zu findePartner
    1.2 Einbindung von SQLite3
    1.3 N.V.

2. Produkt-Einsatz:
    2.1 Das Programm soll von Mitarbeitern einer Firma bedient werden
    2.2 Mitarbeiter (eingewiesen in das Programm)

3. Produkt-Umgebung:
    3.1 Python 3.0 / sqlite3 / tkinter
    3.2 PC
    3.3 N.V.

4. Produkt-Funktionen:
    4.1 Anmeldung von Kunde
            Mithilfe von Benutzername und Passwort wird der jeweilige Benutzer eingeloggt, damit nicht alle daten immmer erneut eingegeben werden müssen
    4.2 Registrierung eies neuen Kunden:
            Ein neuer Kunde wird mithilfe der GUI einfach hinzugefügt
    4.3 Finden passender Partner
            Der passendste Partner wird sofort nach der Registrierung gesucht und dem Mitarbeiter angezeigt.

5. Produkt-Daten:
    5.1 self.name: STR / TEXT (alle ASCII-ZEICHEN)
    5.2 self.vorname: STR / TEXT (alle ASCII-ZEICHEN)
    5.3 self.geburtsdatum: STR / TEXT (alle ASCII-ZEICHEN)
    5.4 self.geschlecht: STR / TEXT ("m"/"w")
    5.5 self.groesse: STR / TEXT (alle ASCII-ZEICHEN)
    5.6 self.figur: STR / TEXT ("sportlich"/"schlank"/"vollschlank"/"dick")
    5.7 self.rauchverhalten: STR / TEXT ("raucher"/"nichtraucher")
    5.8 self.orientierung: STR / TEXT ("heterosexuell"/"homosexuell"/"bisexuell")
    5.9 self.hobby: STR / TEXT (alle ASCII-ZEICHEN)
    5.10 self.adresse: STR / TEXT (alle ASCII-ZEICHEN)
    5.11 self.telefonnummer: STR / TEXT (alle ASCII-ZEICHEN)
    5.12 self.email: STR / TEXT (alle ASCII-ZEICHEN)
    5.13 self.benachrichtigung: STR / TEXT ("telefon"/"email"/"post")
    5.14 self.zahlungsart: STR / TEXT ("lastschrift"/"ueberweisung")
    5.15 self.iban: STR / TEXT (alle ASCII-ZEICHEN)
    5.16 self.kontaktaufnahme: STR / TEXT ("telefon"/"email"/"post"/"chiffre")

    5.17 self.wgeschlecht: STR / TEXT ("m"/"w")
    5.18 self.walter: STR / TEXT (alle ASCII-ZEICHEN)
    5.19 self.wgroesse: STR / TEXT (alle ASCII-ZEICHEN)
    5.20 self.wfigur: STR / TEXT ("sportlich"/"schlank"/"vollschlank"/"dick")
    5.21 self.wrauchverhalten: STR / TEXT ("raucher"/"nichtraucher")
    5.22 self.whobby: STR / TEXT (alle ASCII-ZEICHEN)

6. Produkt-Leistungen:
    6.1 Das Programm soll das ganze so schnell wie möglich abhandeln

7. Benutzungsoberfläche:
    7.1 Anmeldefenster, Hauptfenster (Registrierfenster), Ergebnisfenster
    7.2 normale deutsche QWERTZ-Tastaturbelegung
    7.3 Input -> Pop-Up(Output)

8. Qualitätsziele:
    8.1 Syntaxfehler-freier Quelltext
